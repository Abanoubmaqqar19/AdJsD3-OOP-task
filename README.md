# JavaScript OOP Shapes Project

## Summary

This project demonstrates **Object-Oriented Programming (OOP) concepts in JavaScript** by implementing an **Abstract Shape class**, and its derived classes **Rectangle** and **Square**. It covers:

---
## Concepts
- Abstract classes
- Constructor chaining
- Prototype inheritance
- Accessors (getter/setter)
- Static properties (class counters)
- Method overriding (`toString`, `valueOf`)
- Data descriptors using `Object.defineProperty` and `Object.defineProperties`
